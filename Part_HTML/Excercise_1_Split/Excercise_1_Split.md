# HTML 學習筆記

### Excercise_1：將 JS、CSS、HTML 分開寫。
- object {xxx; xxx;}<br>
針對這物件編輯<br>
- #id {xxx; xxx;}<br>
針對這id的物件編輯<br>
- .class {xxx; xxx;}<br>
針對這類別編輯<br>
一個物件可以用多種class<br>
- *{xxx; xxx;}<br>
HTML內所有的元素做編輯
