# HTML 學習筆記

### Excercise_2 Layout：
- Bootstrap是CSS的框架
- Konva是JS的框架

### Bootstrap 容器：
- container：一般寬
- container-fluid：100%寬
- container-sm：100%寬直到小斷點
- container-md：100%寬直到中等斷點
- container-lg：100%寬直到大斷點
- container-xl：100%寬直到超大斷點

### Grid 網格：
- 最外層為容器，當容器建立好後，內部就可以開始使用網格佈置。
- class="row"：橫著佈置。
- class="col"：直著佈置。
